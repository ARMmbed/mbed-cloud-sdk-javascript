## Prerequisites

[Node.js > v6.0.0](https://nodejs.org), which includes `npm`.

## Installing

After cloning this repository, install the npm dependencies:

```bash
$ npm install
```

## Building

To build and bundle the SDK:

```bash
$ npm run build
```

## Watching (Unit Tests)

```bash
$ npm run watch
```

## Watching (Integration Tests)

```bash
$ npm run watch:integration
```
